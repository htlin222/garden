    # licorice

    (glycyrrhizic acid inhibits 11β-HSD)

    皮質醇，一種糖皮質激素，與糖皮質激素受體結合。 然而，由於其與醛固酮的分子相似性， 它也在較高濃度下與鹽皮質激素受體結合。 醛固酮和皮質醇對鹽皮質激素受體有相似的親和力； 然而，循環中的皮質醇遠遠多於醛固酮。

    # 為了防止皮質醇對鹽皮質激素受體的過度刺激

    11β-HSD將具有生物活性的皮質醇轉化為不再與鹽皮質激素受體結合的非活性皮質醇 11β-HSD與腎上腺皮質激素受體共定位。 甘草含有甘草酸和甘草次酸，能抑制11β-HSD，導致鹽皮質激素過多症候群。 皮質醇水平隨之升高

    抑制了可以抑制cortisol的enzyme

    # 皮質醇與鹽皮質激素受體結合產生

    • 低鉀血症
    • 鹼中毒和高血壓（即鹽皮質激素過量）
