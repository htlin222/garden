    # gastrointestinal stromal tumor

    • 流行病學-胃腸道間質瘤 (GIST) 是: 胃腸道 (GI) 最常見的 → 間葉性腫瘤
      • 它們通常表現 為: 胃和小腸的 → 上皮下腫瘤，但可以 ✔ 累及胃腸道的 → 任何部分
      • 胃腸道間質瘤主要發生在 ↣ 老年人中，在 ↣ 兒童和年輕人中較少見。
    • 分子改變: GIST 中常見的 → 分子改變包括 ≡ KIT、血小板衍生生長因子受體 α (PDGFRA) 和琥珀酸脫氫酶 (SDH) 基因家族的 → 突變
      • 其他分子改變 ✖ 不太常見
      • 建議在 ↣ 大多數診斷 為: GIST 的 → 患者的 → 初始評估過程中對突變狀態進行評估，
      • 因 為: 對全身治療 (例如伊馬替尼和其他酪氨酸激酶抑製劑 [TKI]) 的 → 臨床反應與腫瘤基因型相關並 〓 影響治療選擇

    • 臨床表現: 根據年齡和原發腫瘤的 → 位置，胃腸道間質瘤患者可以 ✔ 表現出多種症狀。

      • 成年患者: 大多數上皮下腫塊的 → 成年患者 ✖ 無症狀，其腫瘤 是: 偶然發現的 →
        • 其他人可能會出現非特異性慢性症狀 (例如，早飽或腹脹)
        • 有些腫瘤可能長到足以潰爛、引起疼痛或導致腫瘤破裂、胃腸道出血、腸穿孔或胃腸道梗阻
      • 兒童和 young adult 患者: 兒童、青少年和年輕成人(AYA) 患者通常會出現胃腸道出血、疲勞和貧血
    • 診斷評估: 胃腸道間質瘤 是: 在 ↣ 無症狀患者中偶然發現的 → ，或者 是: 在 ↣ 對有症狀患者的 → 評估過程中發現的
    • 影像學和診斷程序: 影像學 (例如，腹部和骨盆的 → 計算機斷層掃描 CT) 和內窺鏡程序 (例如，上消化道內窺鏡檢查和超聲內鏡 EUS) 對於 GIST 的 → 初步診斷評估都 是: 必要的 → 。
    • 組織取樣: 只要有可能，我們都會進行術前活檢而 ✖ 不 是: 切除術來確認診斷。
    • 診斷: GIST 的 → 診斷 是: 通過組織病理學、免疫組織化學( GIST IHC ) 和疾病特異性分子改變的 → 鑑定來確定 GIST molecular alterations
    • Treatment of unresectable and metastatic GIST 🪴
    • 轉移性疾病的 → 評估 - 對於所有確診 為: GIST 的 → 患者，我們使用腹部和骨盆的 CT 和/或磁共振成像 (MRI) 來評估轉移性疾病
    • 正電子發射斷層掃描 (PET)-CT 或 MRI 可用於更好地顯示肝轉移； PET-CT 也可用於追踪治療反應
    • 對於大小 ≥2 厘米的 → 原發腫瘤，我們還獲得胸部 X 光片或胸部 CT。
    • 風險分層和預後: 已經開發了各種預後模型和風險分層方案來預測轉移性疾病的 → 風險，識別有復發風險的 → 患者，並 〓 使用 TKI 進行全身治療進行個體化管理
    • 這些模型中納入的 → 危險因素包括 ≡ 腫瘤大小、有絲分裂率、原發部位和腫瘤破裂。 (參見上文‘風險分層和預後’)
    • 武裝部隊病理研究所 (AFIP) 預後模型 是: 美國最常用的 → 風險分層方案
